ES6--下一代javascript语言

ECMAScript 6.0（以下简称 ES6）是 JavaScript 语言的下一代标准，已经在2015年6月正式发布了。它的目标，是使得 JavaScript 语言可以用来编写复杂的大型应用程序，成为企业级开发语言。

babel转码器

各大浏览器的最新版本对ES6 的支持度已经越来越高了，超过90%的 ES6 语法特性都实现了。但是考虑到现有环境是否支持,为此还是做了相应的处理,Babel 是一个广泛使用的 ES6 转码器，可以将 ES6 代码转为 ES5 代码，从而在现有环境执行。

配置.babelrc文件,存放在项目根目录下

            
               {
                   "presets": [],
                   "plugins": []
               }

配置package.json文件:$ npm init

            
                "scripts": {
                    "test": "echo \"Error: no test specified\" && exit 1",
                    "build": "babel src/index.js -o dist/index.js"
                },
                "devDependencies": {
                    "babel-cli": "^6.26.0",
                    "babel-preset-es2015": "^6.24.1"
                }

$ npm run build 完成ES6 => ES5的编译

变量声明:let 和 const

使用let和const声明的变量特性:

1- 只在声明所在的块级作用域内有效 

            
                function testLet(){
                    {
                        let a = 1;
                        var b = 2;
                    }
                    console.log("b="+b);
                    console.log("a="+a);   
                }

2- 不可重复声明, 若在同一代码块中声明变量,会造成js运行异常

3- 使用const声明的变量是一个只读的常量, 一旦声明, 常量的值就不能改变,否则,同上

变量的解构赋值

模式匹配 -- 只要等号两边的模式相同，左边的变量就会被赋予对应的值

            
                完全解构
                function pattern() {                  
                    let a, b, rest;
                    [a, b, ...rest] = [1,2,3,4,5,6];
                    console.log(a,b,rest);                                   
                }

                不完全解构
                function pattern1() {
                    let a, b, c;
                    [a, b, c] = [1,2,3,4,5,6];
                    console.log(a,b,c);  
                }

解构赋值 --交换变量的值(用途)

            
                let x = 1;
                let y = 2;
                
                [x, y] = [y, x];
                console.log("x="+x, "y="+y)

解构赋值 --提取JSON数据(用途)

            
                let jsonData = {
                    id: 42,
                    status: "OK",
                    data: [867, 5309]
                };
                      
                let { id, status, data: number } = jsonData;    

                上述等同于  let { id:id, status:status, data:number } = jsonData

                console.log(id, status, number);

字符串扩展

这里仅强调最常用到的字符串模板 --极大简化字符串和变量拼接

            
                let user={
                    name:'list',
                    info:'hello world'
                };

                console.log(`i am ${user.name},${user.info}`);

如果使用模板字符串表示多行字符串，所有的空格和缩进都会被保留在输出之中。

            
                const A = 'first';
                const B = "second";
                let str = `< ul>
                                < li> ${A} < /li>
                                < li> ${B} < /li>
                            < /ul>`
            
                console.log(str)

模板字符串中嵌入变量，需要将变量名写在${}之中。大括号内部可以放入任意的JavaScript表达式，可以进行运算，以及引用对象属性。模板字符串之中还能调用函数。-- `foo ${fn()} bar`

函数的扩展

ES6 允许为函数的参数设置默认值，即直接写在参数定义的后面。

            
                function log(x, y = 'World') {
                    console.log(x, y);
                }
                
                log('Hello') 
                log('Hello', 'China') 
                log('Hello', '')

与解构赋值默认值结合使用

            
                function foo({x, y = 5}) {     

                    // {x,y=5} == {x:x, y:y=5}
                    // 前面的x,y只是为了与键名匹配, 后面的x,y是输出的值

                    console.log(x, y);  
                }
                    
                foo({})            
                foo({x: 1})         
                foo({x: 1, y: 2})   
                foo()

rest 参数搭配的变量是一个数组，该变量将多余的参数放入数组中。

            
                function test3(...arg){
                    for(let v of arg){
                        console.log('rest',v);
                    }
                }
                test3(1,2,3,4,'a');

数组的扩展

扩展运算符（spread）是三个点（...）。它好比 rest 参数的逆运算，将一个数组转为用逗号分隔的参数序列。

            
                function add(x, y) {
                    return x + y;
                }
                    
                const numbers = [4, 38];
                add(...numbers)
